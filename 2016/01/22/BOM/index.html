BOMbrowser object model 浏览器对象模型

属性

属性名 描述
navigator 浏览器信息
location 浏览器定位和导航
history 窗口浏览历史
screen 屏幕信息

window.navigation浏览器信息

  1. window.navigation.platform当前运行系统信息
  2. window.navigator.userAgent浏览器信息,可以用来判断当前用户所运行的是什么浏览器

window.location浏览器链接

它是一个对象，代表浏览器的定位与导航

  • protocol 协议
  • hostname 主机名
  • port 端口号
  • host 主机地址
  • pathname 资源地址
  • search 查询字符串
  • hash 哈希部分
  1. window.location.href返回完整的页面路径，以通过修改它来完成页面跳转
  2. window.location.searchurl地址?后的内容
  3. window.location.hashurl地址#后的内容
  4. window.location.assign(url)载入新的url,原页面记录浏览历史
  5. window.location.replace(url)载入新的url，原页面不记录浏览历史
  6. window.location.reload() 重新载入页面

history浏览历史

  1. history.back() 传入正整数，表示要后退的步长
  2. history.foward() 传入正整数，表示要前进的步长
  3. history.go() 传入整数，表示跳的步长

screen屏幕信息

  • width 屏幕宽度
  • height 屏幕高度
  • availHeight 浏览器可用高度（屏幕高-系统底栏高度）
  • availWidth 浏览器可用宽度

窗口尺寸

可视区尺寸

  1. 宽 document.documentElement.clientWidth
  2. 高 document.documentElement.clientHeight

滚动距离

  1. document.documentElement.scrollTop[scrollLeft] 非chrome下
  2. document.body.scrollTop[scrollLeft] chrome因为浏览器的滚动条距离应该是属于body下的

内容尺寸

  1. scrollHeight[scrollWidth]元素实际占据的内容的大小，其实就是除了边框以外，内里的内容从顶部到底部的距离

文档高度

  1. document.body.offsetHeight 获取整个文档内容的高度

window下的方法

方法名 描述
alert(),comfirm(),prompt() 对话框
setTimeout(),setInterval() 定时器
open(),close() 打开关闭窗口

BOM下的大多数方法都是在window下的，而js下规定，如果方法是window下的，可以不写如window.open()可以写成open()

对话框，页面会阻塞浏览器后面线程执行

alert() 警告框

comfirm() 确定框

点“确定”，返回的是true,点“取消”返回的是false

prompt() 输入框

用户点击确定后，会返回用户在输入框输入的内容

open(网页url,打开的方式target)打开一个新窗口

  1. 如果url为空，默认为空
  2. 如果target为空，默认为新窗口打开
  3. 它会返回新窗口的window对象

close()关闭指向的window

  • ff:无法关闭当前窗口
  • chrome:关闭当前窗口
  • ie:让用户确定是否关闭
  • 可以关闭通过js打开的新窗口

事件

事件名 出发时机
load 文档和所有图片加载完成时
unload 当离开当前文档时
beforeunload 与unload类似，但它提供询问用户是否确认离开的机会
resize 浏览器窗口大小发生变化时
scroll 拖动滚动条时

onload 文档和所以图片加载完成时会触发

onscroll当滚动条滚动时

onresize当窗口大小被调整时

这两个都是按时间来算的，比如说，它们都是在5毫秒里检测一次，但在5毫秒里拖动了多次，那依然只是会出发一次
