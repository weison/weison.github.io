作用域

发表于 2016-02-12 |

作用域概念

在特定的空间或范围内对数据进行读写操作

  1. 作用：读、写操作
  2. 域：空间、范围、区域…

一对script标签就是一个域

  • 有域的地方就会发生预解释和逐行解释代码的过程

  • 在解释的时候，是自上而下来解读的，一个处理完再处理另外一个

  • 自上而下的解释过程中，上一个全局script的变量与函数在下一个全局script中仍然可以被读取写入

一个函数是一个局部作用域

作用域链

函数内部也算一个作用域，当在预解释过程中，在函数里面没有找到a的变量值时，会从子作用域(fn1)跳到父级作用域查找，这种从里到外的查找的过程叫作用域链

一个参数就相当于一个局部变量

js解释过程

1.预解释 ‘找一些东西’var function 参数

  1. 把声明的变量找出来并默认值设为undefined
    a=undefined
  2. 所有的函数声明，整个提取出来
    fn1 = function fn1(){alert(1);
  3. 遇到重名的情况
  • 当变量与函数重名,函数覆盖变量（因为变量值是undefined，
    而函数是整个函数块，所以会采用函数）
  • 函数与函数重名时，后面的的覆盖前面的

2.逐行解读代码

表达式 + - * / % Number() 参数，所有能改变变量值的操作都叫表达式，它是可以改变预解释的值的。

函数外部读取函数内值的方法

把全局变量放进函数内，然后把想读取的值赋予这个全局变量

局部函数调用(把全局函数放到函数内调用)

判断是不是作用域，就看有没有预解释的过程

if``for``do while都不是作用域

要注意Firefox下不能对{}内的函数进行预解释,所以为了达到兼容的目的，尽量不要在if里定义全局函数和全局变量
