运算符

算术 + 加

  1. 要注意加号在运算时，当有其中一个值是字符时，会将结果隐性转换成字符串

- 减

* 乖

/ 除

% 取模

0%3 =0
1%3 =1
2%3 =2
3%3 =0
利用取模的当除以一个与除数相同的数时等于0（3%3=0）的特征进行隔行变色,利用鼠标移入记录颜色值，移出时将记录的值还原给li的方法来实现移入变色，移出还原
记录index的值，判断index的值是否等于被记录的beCheck值来分配不同的颜色

关系

> 大于

< 小于

>= 大于等于

<= 小于等于

== 数值相等

!= 数值不等

=== 数值及类型相等

!=== 数值和类型不等

  1. 只比较值 == != ‘200’ == 200 //=> true
  2. 比较值和数据类型 === !=== ‘200’ == 200 //=> false

逻辑

&& 与(所有条件都要成立)

  1. 从左到右判断，两边都成立，才会返回true

|| 或(其中一个条件都要成立)

! 否(相反)

  1. 将true变成false

  2. 将数据隐性转换成boolean
