发表于 2016-02-11 |

判断

if

switch 多件事件挑一件来做

  1. break 让代码跳出switch语句，如果忽略break关键字，就会导致执行完一个case继续执行另一个case;
  • default当所有情况都不成立时所使用的默认值
  1. 如果确实需要同时混合几种情况，建议添加注释，说明是有意忽略break的
  • switch()可以使用在任何数据类型中，无论是数字，字符串，还是对象都没有问题。
  1. 使用在布尔值下
  1. 使用在字符串下
  • switch()的判断是全等执行(即数据类型和值都相等)的，不会触发类型转换

? :三目运算

判断条件 ? 条件成立执行的语句 : 不成立执行的语句

循环

for

  1. 设置初始值(var i=0;)
  2. 设置循环执行条件(i<3)
  3. 执行循环体中的内容({console.log(i);})
  4. 每轮循环后都执行操作(i++)
  5. 注意当循环遇到break或continute时，后面的代码都不会执行（包含i++都不执行）

while

for...in

用来循环一个对象的属性名和属性值的
数字也可以当属性名，可以通过obj[number]来访问,同时在for循环时，它会先循环数字类型的属性（从小到大），接下按我们定义好的顺序

跳出循环break与跳过循环continue

跳出循环break指的是不再进行循环

跳过循环continue指的是跨过本次循环，进行下一次循环

label为代码添加标签。

配合break或continue来使用可以的循环中跳出到特定位置

数据类型的真假

数据类型有：

  1. 数字(包含NaN )
  2. 字符串
  3. 布尔值
  4. 函数
  5. 对象(element,[],{},null)
  6. undefined

真

  1. 所有非0的数字(1,-1,1231,1.231)
  2. 所有的非空字符串(‘afbd’，’不可思异’,’ ‘有空格)
  3. true
  4. 所有的函数(function(){})
  5. 能找到的元素
  6. [],{}数据与json就像一个仓库，无论内面有没有放东西，
    这个仓库都真实存在，所以永远返回true

假

  1. 0、NaN
  2. 空字符串(‘’无空格)可以理解为空字符串就像在放屁一样，
    但里面包含了空格，就是想说但没说，所以会返回true
  3. false
  4. 不能找到的元素
  5. null
  6. undefined 未定义
