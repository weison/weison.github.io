页面布局

发表于 2016-04-02 |

所谓布局就是浏览器以正确的大小在内容放在正确的位置上。

影响布局的属性

display呈现方式

  1. 设置元素的显示方式
  2. display:block|inline-block|inline|none
属性 特征 默认元素
block 可设宽高，默认宽度为父级元素的宽度 独占一行 div,p,h1-h6
inline 设置宽高无效，宽度为内部内容的宽度，文本流形式 a,span,img
inline-block 可设宽高，默认为内部内容的宽度，文本流形式,注意假如inline-block内部内容的宽度加上这行内容的宽度超过整行的宽度时，inline-block元素会独占一行 input,textarea,select,button
none 设置元素不可显示

display:none与visibility:hidden的区别

  1. display为none时，相当于脱离了文本流，后面的元素会占据它原来的位置
  2. visibility：hidden，它只是隐藏了内容，但还是会占据着它原来的位置

    position定位

  3. 通过top,right,bottom,left设置元素边缘距离参照物的距离，z-index设置层级
  4. 当同时设置2个相对的值时而且未设置宽高值的情况,如top和bottom值时
    |relative|会根据先后定位顺序来覆盖|
    |absolute或fixed|会改变元素的尺寸直到符合规则|
    而如果设置了宽高，则按定义的顺序来，如先定义left,后定义right,则使用right

    relative 相对定位

    相对于其自身的原来位置的距离

absolute 绝对定位

相对于其最近定位了positon:absolute|fixed的元素的距离

fixed 固定定位

相对于浏览器来进行定位

sticky 粘性布局
