基本数据类型

Number数字类型

包含了正数、负数、0，小数，NAN(not a number)不是一个有效的数字，但他还是属于number数据类型的，而且NAN == NAN是fasle

Number()强制转换为数字类型

要求待转换的数据一定都是都是数字，否则会返回NaN

paerInt()/parseFloat()非强制性数据转换

从左到右，逐个查找字符串，中途遇到非有效数字，就不再继续查找

isNaN()

检测一个数字是否有效数据

  • true 不是有效数字
  • false 是有效数字

Boolean 布尔类型

! 取反，首先将类型转换成Boolean类型的，然后再取反
!! 将其它数据类型转换成Boolean类型，相当于Boolean()

一个值(!val1)进行判断时为false的情况

  • 0
  • “” 空字符串
  • null
  • undefined
  • NaN

两个值( val1 == val2 )进行 == 比较时

假如数据类型不一样，默认会进行数据类型转换

对象对比时，永远不相等(因为是声明了两个不同的数组，会存放在不同的内存位置上)
对象与字符串进行比较，先调用对象的toString()转换成字符串，再进行比较
  1. ([].toString()) => "" 空数组转换成字符串，会变成空字符串
  2. ({}.toString()) => "[object Object]" 空对象转换成字符串，会变成 "[object Object]"
对象与布尔值进行比较
  1. 先调用对象的toString()方法，把对象转换成字符({} => “[object Object]” ,[] => “” )
  2. 再把字符串调用Number()方法，转换成数字(空字符串转换成数字，结果为0,Number(""); //=> 0)
  3. 然后再布尔值转换成数字，true转换成1，false转换成0
  4. 最后再进行比较
对象与数字进行比较
  1. 先调用对象的toString()方法，把对象转换成字符
  2. 再把字符串调用Number()方法，转换成数字(空字符串转换成数字，结果为0,Number(""); //=> 0)
  3. 最后再进行比较
数字与布尔值进行比较
  1. 布尔值转换成数字，true转换成1，false转换成0
  2. 进行比较
数字与字符串进行比较
  1. 字符串转换为数字（Number()）
  2. 进行比较
布尔值与字符串进行比较
  1. 都转换成数字再进行比较
null与 undefined进行比较为true
null或 undefined与其它数据进行比较，都为false

两个值( val1 === val2 )进行 === 比较时,会进行绝对比较

只有是类型不相等都为false,它不会进行数据类型转换

创建方式

删除属性

  2. delete

js中，对象（不是指object）、类、实例的区别

  • 对象就泛指，js中一切皆对象
  • 类，是对象下的一个细分
  • 实例，是类中一个具体事物

基本数据类型与引用数据类型的区别

基本数据类型操作的是值，数据类型操作的是内存地址引用

  • 基本数据类型是值的传递，值的修改不 会影响到其它变量

  • 引用数据类型是值的引用（可相当把内存地址标识赋给了它），值修改会影响引用这个地址的所有变量

  1. 定义一个变量
  2. 开辟一个新的空间，然后把属性名和属性值保存在这个空间中，并且有一个空间地址
  3. 把空间地址给了这个变量，变量并没有储存这些数据，而且储存了这个空间引用地址
  4. 如果过程中把这个地址也给了这个变量，另一个变量也是存储这个地址，此时两个变量操作的是同一个空间地址
  5. 其中一个改变了，另外一个也跟着改变

函数funciton

定义一个函数的步骤

  1. 开辟一个新的空间地址
  2. 把字符串的js代码当作是字符串存到了空间里面（一个函数如果只是定义了没有执行的话，这个函数没有任何意义）
  3. 把函数地址给函数名

执行一个函数

  1. 首先会形成自己的一个独立作用域
  2. 把定义时候当成字符串的代码当js代码来执行

形参

  • 可以理解成是一个变量，它的作用是，在我们定义函数时，无法知道参数的实际内容是什么，因此采用形参的方式来代表需要处理的数据内容，当执行这个方法时，只要把实际数据代表形参即可
  • 定义一个形参，假如没有传值，那个这个形参的默认值为undefined

多态

一个功能的多种形态，通过参数的不同实现不同的功能

arguments是函数内置的接收参数的机制

  • 系统自带的，不管是否传递过参数，也不管有没有定义的形参，arguments一直存在
  • 这是一个类数组的形式，索引值从0开始代表第几个传递的参数
  • arguments.length代表传递参数的个数

arguments.callee 代表当前函数本身

return

  • 如果想获取函数内部的值，可能采用return的方法来把结果返回出去。
  • 如果函数没有写return内容，或者直接写return,那么系统默认返回undefined
  • 在函数体内，return后面的语句不会被执行，利用这个特征，可以控制函数体在哪个中止执行

数组

数据是由多个项组成的，每一组之间用逗号隔开，并且每一项可以是任务类型的数据。它也是对象的一种，对象数据类型是由属性名：属性值组成的。数据的属性名是它的索引值

创建方法

  1. 字面量
  1. 实例方式创建

数组的常用方法

数组的增删改
push

在数组的后面增加内容
返回的是新数据的长度
原来的数组内容改变了

unshift

在数组的前面增加内容
返回的是新数据的长度
原来的数组内容改变了

splice
  1. splice(n,0,x)向数组中的某个位置添加内容，从索引值n开始，删除0个内容，把新增加的内容放在索引n的前面,返回的是一个空数组
    原来的数组改变了
  • n 开始的索引值，添加到哪个索引值前面
  • x 增加的内容
  1. splice(n,m)从索引值n（包含n）开始，删除m个元素，把删除的内容当做一个新的数组返回，原有数组改变
  1. splice(n,m,x)替换数据中的内容，先删除再添加（从n开始，删除m个元素，然后插入X）
pop()

删除数组的最后一项，返回被删除的元素
原数组会被改变

concat(x)

实现数组的拼接,原有数组不变,假如不填写x,就会变成数组的复制

数组的转换成字符串
toString()

把数组的每一项拿出来用逗号隔开，原有数组不变

join(分隔符)

把数组的每一项拿出来用分隔符隔开，原有数组不变

也可以利用它和eval来进行纯数字数组的运算,eval的作用是把字符串转换为可执行的表达式

数组的排列和排序

冒泡排序

冒泡排序的思想是当前项与后一项进行比较，假如当前项比后一项大，那就交换位置

  1. 轮数 = arr.length - 1
  2. 每一轮的次数 = arr.lenght -1 - 当前轮数
  3. 交换位置的方法是建立一个临时变量储存当前值
