bower

相对于npm来说，bower更偏向是一个前端的包管理工具

  1. npm install bower -g 安装bower
  2. bower install <package>下载指定的包
  3. bower uninstall <package>删除指定的包
  4. bower info <package> 查看指定的包版本信息
  5. bower ..

标签属性和条件渲染

绑定标签属性 v-bind

通过v-bind来绑定的标签属性是动态绑定的，当这个属性的数据被修改时，页面也会动态地更新这个属性

常用的功能是用来绑定class

  • 可以是直接的字符串

  • 可以是对象，当class为对象时，class的生效由value是否为true来决定

  • 可以是数组，那么这些，class的名称指向了data里面的属性

  • 可以混合一起用,数组加对象

也可以用变量的方式来渲染内联样式

组件之间通信Vuex

是什么

Vuex 是一个专为 Vue.js 应用程序开发的状态管理模式。它采用集中式存储管理应用的所有组件的状态，并以相应的规则保证状态以一种可预测的方式发生变化。

State

Vuex 使用 单一状态树 —— 是的，用一个对象就包含了全部的应用层级状态。至此它便作为一个『唯一数据源(SSOT)』而存在。这也意味着，每个应用将仅仅包含一个 store 实例。单一状态树让我们能够直接地定位任一特定的状态片段，在调试的过程中也能轻易地取得整个当前应用状态的快照。

Getters

当需要从State中派生出一些属性时，可以使用Getters,可以理解为Vuex中的computer方法,vue2.0官方建议不要使用箭头函数，因为箭头函数是指向上一级
