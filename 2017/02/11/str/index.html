字符串操作

发表于 2017-02-11 |

javascript中的对象是一个泛指，所有的东西都是对象，类是指把对象具体细分Number、String、Function、Object、Date、RegExp，实例是类中的一个具体东西，如{},[],’123’;

length获取长度(空格也会计算)

charAt根据索引值获取对应字符

charCodeAt获取字符对应的unicode编码

每一个字母、汉字、特殊字符，都有它对应的unicode码值

String.formCharCode 根据unicode编码，返回对应字符

indexOf从左到右查到字串第一次出现的索引值

str.indexOf(searchValue[, fromIndex])

  • searchValue 一个字符串表示被查找的值。
  • fromIndex 表示调用该方法的字符串中开始查找的位置。可以是任意整数。默认值为 0。如果 fromIndex < 0 则查找整个字符串（如同传进了 0）。如果 fromIndex >= str.length，则该方法返回 -1，除非被查找的字符串是一个空字符串，此时返回 str.length。
  • 区分大小写的
  • 假如没有查找到内容，会返回-1
  • fromIndex < 0 则查找整个字符串（如同传进了 0）
    String.prototype.indexOf()

search()搜索字符串，并返回子字符的索引值

lastIndexOf() 从右往左查找符合要求的字符索引值

字符串对比（相当于字符unicode编码对比）

str.substr(start[, length])截取

返回字符串中从指定位置开始到指定长度的子字符串。

  • start
    开始提取字符的位置。如果为负值，则被看作 strLength + start，其中 strLength 为字符串的长度（例如，如果 start 为 -3，则被看作 strLength-3）。
  • length
    可选。提取的字符数。
    假如start为负数，且abs(start)的长度大于str.length,则截取会从0开始

String.prototype.substr()

str.substring(indexStart[, indexEnd])

返回字符串两个索引之间（或到字符串末尾）的子串。
String.prototype.substring()

  • 如果任一参数小于 0 或为 NaN，则被当作 0。

slice截取

str.slice(beginSlice[, endSlice])提取字符的一部分，并返回新的字符串
String.prototype.slice()

  • slice 不修改原字符串，只会返回一个包含了原字符串中部分字符的新字符串
  • slice() 提取新字符串不包括 endSlice。

跟substring有两点不同：

  1. 不同根据两个数字的不同而调换位置

  2. 当索引值是负数时，计算方式为str.length + 索引

通过截取来隐藏部分内容并添加省略号

toUpperCase和toLowerCase 英文大小写转换

  1. toUpperCase() 小写英文转换成大写
  2. toLowerCase() 大写英文转换成小写

replace(旧字符，新字符)

  • 在不使用正则的情况下，每次执行，只会替换一个

split 将字符折分成数组

str.split([separator][, limit])通过把字符串分割成子字符串来把一个 String 对象分割成一个字符串数组。
String.prototype.split()

  • 不写任何参数，表示不进行拆分，直接将整个字符串转换为数组；
  • 填写””即将每个字符将分成一个数组；
  • 假如带参数，则将参数当划分线，划分后，被当作分割符的内容不保留

join 将数组转换成一个字符串

  1. 生成随机验证码
